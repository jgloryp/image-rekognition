카카오 비전 API (성인 이미지 판별)
https://developers.kakao.com/docs/latest/ko/vision/dev-guide#recog-adult-content


반환 키 및 밸류 설명
soft:	이미지에 약간의 노출이 있을 확률, 주로 수영복 사진일 경우 해당, (Float 0 ~ 1.0 사이의 값)
adult: Float	이미지가 노출이 많은 성인 이미지일 경우, 주로 음란물일 경우 해당, (Float 0 ~ 1.0 사이의 값)
normal:	Float	이미지가 정상일 확률, (Float 0 ~ 1.0 사이의 값)


가격
5천개/일 무료 (가격이 공개되어 있지 않음, 사용량에 따라 추후 계약 체결 방식)
